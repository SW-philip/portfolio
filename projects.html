Waybar · NixOS

The personality layer

A voice applied to every system state on my desktop

Active

My Waybar setup isn't a status bar — it's a character. Every system state has a corresponding line: battery levels, network quality, CPU temperature, volume, weather, media. Each one has a rotating set of responses that share a consistent voice. Not helpful, or even informative.

Only opinionated.

The snark JSON has grown to cover fourteen system states with eight to thirteen lines each. The CPU at critical temp: "The fans scream into the void." Ethernet connected: "Tethered to truth at 1Gbps." Fog outside: "The world is buffering." It's the same voice every time. It just happens to live in a status bar.

Quantum Clock

Time as a narrative, not a number

Active

Five different ways to display the time, rotating or selectable. The interesting one: the caliper clock. Tick marks flank a labeled object. From midnight to noon, the objects grow — a seed, a stone, a chestnut, a book. From noon back to midnight, they shrink through a different set of objects entirely. The clock tells a story. It just happens to be accurate.

▓▓▓░░░░░░░ 34% 8:18 AM Day progress — how far through the day you are
twenty past eight Natural time — plain English, no numerals
——| a chestnut |—— 8:18 Caliper clock — an object growing and shrinking through the day
🌗 43% illuminated 21d to full Moon — lunar phase and days to next event

sqlch

A radio streaming tool that knows what it's playing

Repo coming soon

Command-line radio streaming with a Textual TUI, Unix socket daemon, and MPRIS integration — so your desktop environment knows what's playing and can control it. Station metadata gets enriched through Spotify and MusicBrainz, so ICY stream titles resolve to actual artist and track information where possible.

It's fully functional. I'm doing a reproducibility pass before the repo goes public — I want it to actually work when someone else clones it, not just on my machine.

Unix socket daemon — runs in the background, survives terminal closes

MPRIS integration — play, pause, skip from any MPRIS-aware controller

ICY metadata — parses stream titles in real time, handles edge cases

Spotify + MusicBrainz enrichment — resolves stream metadata to real artist info

Textual TUI — full terminal interface with station search, preview, and edit

Waybar module — pipes now-playing into your status bar
