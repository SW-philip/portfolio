Theme Registry · Eval-time Discovery Nix Meta-programming

A theme registry that builds itself at nix eval with no manifest files and no code generation. builtins.readDir walks the theme tree during evaluation, discovering every family and variant from directory structure alone. Palette files are imported live; their values are immediately available as typed Nix expressions — colors, shell exports, and light/dark heuristics all resolved before a single derivation is instantiated.

home/niri/default.nix
allThemes =
  let
    loadTheme = family: folderName:
      let
        dir    = "${themesRoot}/${family}/${folderName}";
        files  = builtins.readDir dir;
        nixName = builtins.head (builtins.attrNames (
          lib.filterAttrs (n: _:
            lib.hasPrefix "palette-" n && lib.hasSuffix ".nix" n
          ) files));
        slug = lib.removeSuffix ".nix"
                 (lib.removePrefix "palette-" nixName);
        p    = import "${dir}/${nixName}";
      in {
        inherit family slug dir;
        palette   = p;
        shContent = builtins.readFile "${dir}/${shName}";
        isLight   = p.isLight or
                      ((p.FONT_SIZE_BAR or "12px") == "13px");
      };
    familyThemes = family:
      lib.mapAttrs' (folder: _:
        lib.nameValuePair
          (loadTheme family folder).slug
          (loadTheme family folder))
        (lib.filterAttrs (_: t: t == "directory")
          (builtins.readDir "${themesRoot}/${family}"));
  in
    lib.foldlAttrs
      (acc: family: _: acc // familyThemes family)
      {} familyDirs;
Surface Kernel · structuredExtraConfig Custom Kernel Build

The Surface Pro 7+ ships no upstream drivers for its hardware aggregator bus, touch/stylus stack, or IPU3 camera pipeline. This Nix expression passes structured kernel configuration — typed module values against Nix's kernel type system rather than raw .config strings — building the complete linux-surface module set into a custom kernel derivation at flake eval time.

pkgs/linux-surface-6_19/patches.nix
structuredExtraConfig = with lib.kernel; {
  # Surface hardware aggregator bus
  SURFACE_AGGREGATOR               = module;
  SURFACE_AGGREGATOR_BUS           = module;
  SURFACE_AGGREGATOR_REGISTRY      = module;
  SURFACE_AGGREGATOR_TABLET_SWITCH = module;
  SURFACE_DTX                      = module;

  # Touch / stylus input stack
  HID_IPTS                         = module;
  HID_ITHC                         = module;

  # IPU3 image signal processor (front camera)
  VIDEO_IPU3_CIO2                  = module;
  VIDEO_IPU3_IMGU                  = module;
  VIDEO_OV5693                     = module;

  # Thermal / power sensors
  SENSORS_SURFACE_TEMP             = module;
  SENSORS_SURFACE_FAN              = module;
  BATTERY_SURFACE                  = module;
};
