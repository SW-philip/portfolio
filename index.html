How I work

01

Build for the integration you don't have yet

Every decision gets made as if the API is coming tomorrow. When it arrives, you swap one thing, not everything.

02

Let the conversation resolve ambiguity

Don't ask for clarification upfront. Design the first pass to cover the most likely intent, then escalate if needed. Progressive disambiguation.

03

Know when the answer is a human

A bot that recognizes its own ceiling is more valuable than one that tries to handle everything. Triage is a design decision.

04

No dead ends

A member should always have a path forward. A representative is reachable from anywhere, through natural language, without a button.
